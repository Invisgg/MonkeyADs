# MonkeyADs
- Minehut Auto Advertisement System
- Perks:
Walks to random location
Random Advertisment delays
Doges advertisements.

# How to use
- NodeJS:
1. install npm/node
2. run `npm install`
3. run `node index.js`

- PKG (exe):
1. install npm/node
2. run `npm install`
3. run `npm install pkg`
4. run `pkg index.js`

# More Info
